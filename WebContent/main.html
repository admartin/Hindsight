Welcome to Hindsight!

Created by Team Ninja Turtles

The goal of this project was to create a twitter extension that allowed for a user to search any public twitter profile and view basic statistics not otherwise provided to a user by twitter. Using a complex parsing algorithm, Hindsight sorts the tweets of the provided twitter profile and places them into categories. The user is then able to compare the twitter profile's average tweets in each category to the overall twitter community average. The overall average was compiled by running our complex parsing algorithm over more than 30,000 random tweets pulled from the twitter community.
